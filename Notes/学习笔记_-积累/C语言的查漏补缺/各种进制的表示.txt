====== 各种进制的表示 ======
Created 星期二 07 六月 2022

在编程时，想要输入一个非十进制的数。
各种进制的表示：
	
	二进制：前面添加前缀0b (0B)
		int a	= 0b101010110;
	
	八进制：前面添加前缀0
		int a = 0755;
		
	十六进制：前面添加前缀0x (0X)
		 int a = 0x dab;
